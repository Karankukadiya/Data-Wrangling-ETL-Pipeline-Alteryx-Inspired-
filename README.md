# Data Wrangling & ETL Pipeline (Alteryx Inspired)

## 📌 Overview
This project demonstrates a data wrangling workflow similar to Alteryx. It cleans and joins three datasets (sales, customer, product) into a single analysis-ready dataset.

## 🎯 Objective
- Clean raw datasets
- Remove duplicates and missing values
- Join multiple datasets
- Automate data preparation

## 🛠️ Tools Used
- Python (Pandas)
- Data Cleaning Techniques
- Data Transformation
- Data Joining

## 📂 Datasets
1. Sales Data
2. Customer Data
3. Product Data

## ⚙️ Workflow Steps
1. Load datasets
2. Clean missing values
3. Remove duplicates
4. Standardize data formats
5. Filter invalid records
6. Join datasets using keys
7. Generate final dataset

## 📊 Output
- Cleaned and joined dataset ready for analysis
- Workflow summary file

## 🚀 Key Results
- Reduced manual data preparation time by 80%
- Automated data cleaning and transformation
- Improved data quality and consistency
